## Response Envelopes
All our responses follow the same pattern. They are sent respecting a default response envelope as shown bellow.

```JSON
# Default Envelope
{
    "status": 404,
    "version": "1.0.0"
}

# Envelope with optional error codes
{
    "version": "1.0.0",
    "status": 400,
    "errors": [
        {
            "errorCode": "PCE001",
            "message": "The field 'name' is mandatory."
        }
    ]
}
```
